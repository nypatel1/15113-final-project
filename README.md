# 15-113 final project — AI Posture Coach

Browser-based posture coach that uses computer vision to score your posture in real time and tracks your history across sessions.

The full project lives in [`posture-coach/`](./posture-coach). Start there for setup, architecture, and an explanation of the posture analysis approach.

Quick start:

```bash
cd posture-coach
npm install
npm run dev
```
